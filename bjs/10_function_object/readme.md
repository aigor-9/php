# Проект: Генератор случайных пользователей.

Разработано в качестве практического задания на курсе «Fullstack веб-разработчик на JavaScript и PHP». Модуль — Базовый JS. Функции и объекты

## Используемые технологии

- HTML
- CSS
- JavaScript

## Цель проекта

Создание генератора случайных пользовательских данных: ФИО, пола, даты рождения и профессии.

## Результат

Получилось создать генератор для фамилии, имени, отчесмтва, пола, даты рождения и профессии, с применением в коде, объектов и функций.

## Дальнейшие планы

Далее по плану будет изучение циклов, и применение их в последущих проектах.
