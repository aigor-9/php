# Проект - игра, в которой компьютер угадывает число

Разработано в качестве практического задания на курсе «Fullstack веб-разработчик на JavaScript и PHP». Модуль — Базовый JS. Условия

## Используемые технологии

- HTML
- CSS
- JavaScript

## Цель проекта

Создать игру, в которой компьютер угадывает число пользователя

## Результат

Для создания игры, были применены условий в коде. Получилось создать отображение вопросов не менее чем в трех вариантах, сообщение при успехе не менее чем в трёх вариантах.
При вводе текста, который не может быть интерпретирован как число (NaN), было присвоено значение по умолчанию.
При вводе максимума или минимума больше 999 или меньше -999 изменяется число на ближайшую границу.
Число выводится в текстовой форме, если на его запись в текстовой форме требуется меньше 20 символов.
